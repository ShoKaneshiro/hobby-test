# 🌟 習い事さがし！あなたにぴったりの習い事テスト

7つの質問に答えるだけで、102この習い事の中からあなたにぴったりのTop5を見つけてくれるWebアプリです。

## 使い方

1. `hobby-test.html` をブラウザで開く
2. 習い事リスト（102個）を眺める
3. 「テストをスタート」ボタンを押して7問に答える
4. あなたのTop5が表示される！

## 仕組み

各習い事には5つのタグが付いています（例：`体力` `チーム` `競技` `屋外` `勝負`）。

質問の選択肢ごとに「加点タグ（+10点）」と「減点タグ（-5点）」が設定されており、7問の回答をもとに全102個の習い事をスコアリングして上位5つを表示します。

```
スコア = Σ（一致した加点タグ × +10）+ Σ（一致した減点タグ × -5）
```

## 設計上のこだわり

- **タグ数を全習い事で5個に統一**することで、タグ数の多い習い事が有利になる問題を解消
- 「個人」「集中」のような**汎用すぎるタグを廃止**し、識別力のあるタグに置換
- 全4⁷＝16,384パターンの総当たりテストで、**102個すべての習い事がTop5に出現**することを確認済み
